# MiniBaritone Fabric 1.21.11

Небольшой client-side Fabric-мод под Minecraft 1.21.11. Это не полный Baritone, а минимальная реализация под задачу:

- движение по A* пути;
- обход простых препятствий;
- ломание блоков перед собой;
- постановка блоков под ноги/в разрыв пути;
- команды `/mb`;
- добыча дерева в радиусе до 100 блоков;
- сборка через GitHub Actions.

## Команды

```mcfunction
/mb goto <x> <y> <z>
```
Идти к координатам. Бот строит путь, ломает мешающие блоки и ставит блоки, если включена постановка.

```mcfunction
/mb tree
/mb tree <radius>
```
Искать и добывать бревна деревьев. Радиус ограничен от 1 до 100 блоков.

```mcfunction
/mb breakahead true
/mb breakahead false
```
Включить/выключить ломание блоков на пути.

```mcfunction
/mb place true
/mb place false
```
Включить/выключить постановку блоков для прохода.

```mcfunction
/mb status
/mb stop
```
Показать состояние или остановить бота.

## Как собрать через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы этого проекта в репозиторий.
3. Открой вкладку **Actions**.
4. Запусти workflow **Build Fabric mod** вручную или сделай push.
5. Готовый `.jar` будет в артефактах workflow.

## Важно по версиям

В проекте стоят версии под Minecraft `1.21.11`:

```properties
minecraft_version=1.21.11
yarn_mappings=1.21.11+build.1
loader_version=0.17.2
fabric_api_version=0.136.0+1.21.11
```

Если Fabric на момент сборки выпустил другие точные значения для `1.21.11`, поменяй их в `gradle.properties`.  
Для Loom `1.11-SNAPSHOT` нужен Gradle `8.14+`, поэтому в GitHub Actions уже указан Gradle `8.14.3`.

## Ограничения

- Это учебная mini-реализация, не полноценный Baritone.
- Не умеет надежно проходить сложные паркур-маршруты, лаву, воду, порталы, двери, лестницы и нестандартные блоки.
- Добыча дерева ломает блоки из тега `LOGS`, листья не трогает.
- Для постановки блоков нужны обычные строительные блоки в хотбаре.
- Работает как клиентский мод: команды доступны в игре на клиенте.

## Структура

```text
src/main/java/ru/mond/minibaritone/
  MiniBaritoneClient.java  # регистрация команд и tick event
  BotController.java       # логика движения, добычи, команд
  Pathfinder.java          # A* pathfinding
  TreeTask.java            # сканирование деревьев в радиусе
  BlockUtils.java          # ломание, постановка, проверки блоков
```
